HTML-CSS-JavaScript

A continuacion una breve presentacion de cada lenguaje a elegir:

HTML

HTML5 es la última versión del lenguaje de marcado utilizado para crear y estructurar el contenido de las páginas web. Se trata de un estándar ampliamente adoptado que ha revolucionado la forma en que se desarrollan y presentan los sitios web.

CSS

CSS es un lenguaje que describe la presentación y el estilo visual de un documento HTML . Su principal función es separar el contenido de la estructura y la apariencia visual de un sitio web, permitiendo controlar el diseño y el estilo de una página web .

JavaScript

JavaScript es un lenguaje de programación de alto nivel que se utiliza principalmente en el desarrollo web, . Proporciona capacidades interactivas y dinámicas a las páginas web, permitiendo manipular y controlar el contenido, la apariencia y el comportamiento de la misma.
